## Getting Started


### Installation
```bash
    # clone repo
    git clone [FPL BET API](https://github.com/officialyenum/fpl-bet-api.git)

    #install packages
    $ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
